### Installation

Algo-Visualization  requires Pythont3 to run

Dependency 
```sh
$ pip install pygame
```

For Sorting Algorithm
```sh
$ python <file_loc> list
$ python <file_loc> <Algorithm name>
```

For Graph Algorithm
```sh
$ python <file_loc> 
```
